# 簡介

SNM60（Say No to Mouse）是一個開源（Open-source）的鍵盤專案，包含了硬體和韌體的部分。

其最主要的目的就是將滑鼠的功能完全地整合進一個60%鍵盤內，這樣在使用電腦時，手就不用一直在鍵盤和滑鼠上來回切換。
